Accessible Table View

This is the table equivalent of the interactive visual diagram on the homepage. Every cell links to the same detailed content as the visual version, so whichever format works better for you, you have full access to the framework.

- Each row represents a Knowledge Dimension (what students know).
- Each column represents a Cognitive Process (what students do with that knowledge).
- The verb inside each cell is a sample learning objective at that intersection.

Click any verb to see example AI literacy activities. You can also click a row or column header to learn more about that dimension.
Bloom's stAIrcase interactive table. Rows are Knowledge Dimensions (Metacognitive, Procedural, Conceptual, Factual). Columns are Cognitive Processes (Create, Evaluate, Analyze, Apply, Understand, Remember). Each cell contains a verb that represents a learning objective at the intersection of those dimensions. Row and column headers link to detailed information about each dimension. Cells link to example activities and AI literacy integration strategies.
